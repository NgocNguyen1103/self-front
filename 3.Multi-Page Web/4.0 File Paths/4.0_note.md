# Computer file path

## Folder

### Absolute file path (root folder)

**Example:** C:, D:

### Relative file path

**A**{  a  **B**[  b  c  **D**( e ) ]  **N**(  m  c  ) }

**Starting point**: c

**../** to access to some thing in folder A (bigger folder)

ex: ../a

**./** to access to something in the same folder

ex: ./b or ./D/e

**/** to access to folders that have the same heirachy

ex: /N/m
